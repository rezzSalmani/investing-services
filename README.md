## About The Project

<!-- GETTING STARTED -->
## Getting Started

Follow following steps to run the project.


### Installation




1. Clone the repo
   ```sh
   git clone https://github.com/rezzSalmani/investing-services.git
   ```
   
   <br>
   
2. Install NPM packages
   ```sh
   npm install
   ```
   
   <br>
   
8. Start the development server:
   ```sh
   npm run dev
   ```
